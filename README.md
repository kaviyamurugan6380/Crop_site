# Crop_site
Predictive Irrigation System Using IoT and ML that monitors soil moisture, temperature, humiditiy, and rainfall in real time. It automates irrigation, recommends crops & fertilizers via ML algorithms (Random Forest), and detects plant disease using CNN which boosting agricultural productivity while reducing water waste.

# ⚡ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/Crop_site.git
cd Crop_site
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Flask Server

```bash
python app.py
```

---

## 4️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```

# 🧠 Machine Learning Models

### 🌱 Crop Disease Detection
Detects plant diseases from uploaded leaf images using Deep Learning techniques.

### 🧪 Fertilizer Recommendation
Suggests suitable fertilizers based on soil nutrient values.

### 📊 NPK Analysis
Analyzes Nitrogen (N), Phosphorus (P), and Potassium (K) values for better crop growth.
